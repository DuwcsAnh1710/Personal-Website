Những dự án tiêu biểu đã và đang phát triển

Bốn dự án dưới đây thể hiện hành trình kết hợp UI/UX với lập trình web/backend của mình. Mỗi sản phẩm đều xuất phát từ bài toán thực tế sinh viên hoặc nhu cầu cá nhân, được triển khai và duy trì trên GitHub.

StuSmart

Nền tảng quản lý thông tin sinh viên UTH

Web app giúp sinh viên theo dõi lịch học, điểm số và cập nhật học phí. Mình phụ trách nghiên cứu UX, thiết kế UI trong Figma, tạo prototype HTML/CSS và xây dựng backend Express.js kết nối MongoDB để đồng bộ dữ liệu theo tài khoản.

  • Figma
  • HTML/CSS
  • Express.js
  • MongoDB

MultiDriver Hub

Trình quản lý nhiều tài khoản Google Drive

Ứng dụng desktop tích hợp Python (PySide6) và Node.js để quản lý nhiều tài khoản Drive. Hỗ trợ đồng bộ dữ liệu, tìm kiếm nâng cao và báo cáo dung lượng. Mình đảm nhận kiến trúc luồng, UI và tài liệu hoá API giữa hai ngăn ứng dụng.

  • Python
  • PySide6
  • Node.js
  • Express.js
  • Google Drive API

BattleShip

Trò chơi hải chiến luyện thuật toán

Trò chơi BattleShip chạy trên Python console. Dự án giúp mình rèn luyện xử lý ma trận, kiểm tra va chạm tàu và tạo đối thủ AI cơ bản. Phục vụ môn Cơ sở lập trình và luyện tập cấu trúc dữ liệu.

  • Python
  • Game logic
  • Ma trận
